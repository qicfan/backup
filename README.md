### v1.1.0
- [x] 多线程并发上传（加快备份速度，内网基本不担心网络稳定和带宽）
- [x] 大文件分块上传-经测试request.uploadFile上传大文件不会导致内存占用过高，所以不需要分块上传
- [x] 解决从相册导出大文件慢会导致后台任务退出的问题（长时间没有网络传输）-改为只有在应用退后台时才申请长时任务，并且将文件复制采用了多线程处理，基本没有复现过该问题。
